Begin
input variables L1, B1, L2 and B2 holds integers
variables A1, A2, P1 and P2 holds operations
variables bool1 and bool2 holds boolean value

 Areas of two rectangles
 A1 = L1 * B1
 A2 = L2 * B2

 Perimeters of two rectangles
 P1 = 2( L1 + B1 )
 P2 = 2( L2 + B2 )

 If A1 > A2
    bool1 = true
otherwise
    bool1 = false

 If P1 > P2
    bool2 = true
otherwise
    bool2 = false

print bool1 \n bool2
End
